# stones
Вы играете в игру "Камни": игру для двух игроков с двумя наборами камней по  n n и  m m штук. 
Вы играете в игру "Камни": игру для двух игроков с двумя наборами камней по 
n
n и 
m
m штук. С каждым ходом один игрок может забрать следующие комбинации камней:

Взять один камень из любого набора.
Взять два камня из какого-то одного набора
Взять два камня из одного и один из другого.
Когда камень забрали, он выходит из игры. Побеждает игрок, который заберет последний камень. Первый ход за вами.

Вы и ваш оппонент играете оптимально.

Формат ввода
В первой строке содержится два числа nn (≤n≤1000)(1≤n≤1000), mm (1≤m≤1000) (1≤m≤1000) — количество ваших камней и количество камней у вашего оппонента.

Формат вывода
В единственной строке выведите Loыe
Lose, если вы заведомо проиграете, и Win
Win, иначе
Пример 1
Ввод
4 4
Вывод
Lose
Пример 2
Ввод
17 72
Вывод
Win
Пример 3
Ввод
7 5
Вывод
Win
